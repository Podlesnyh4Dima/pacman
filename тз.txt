Проект: Pacman
Авторы: Гаврилова Алиса и Подлесных Дмитрий

Многие из нас знают старую аркадную игру "Pacman" и мы решили создать ее с помощью библиотеки pyGame.
Наш проект будет реализован с помощью 5-ти классов:
	1. class Main()
	2. class Map()
	3. class Player()
	4. class Enemy()
	5. class EacterEgg()
В каждом классе будут функции, которые будут делать наш код более красивым и удобным.
Например, в классе Player() будут функции:
	1. def draw(self...)
	2. def move(self...)
	3. def death()
	4. и т.д.
